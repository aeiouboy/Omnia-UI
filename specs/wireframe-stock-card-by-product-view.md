## Field Specifications

| Field | Type | Description |
|-------|------|-------------|
| Date & Time | DateTime | Transaction timestamp in GMT+7 (format: MMM DD, YY HH:mm:ss) |
| Transaction Type | Enum | Type of stock movement with color-coded badge |
| Quantity | Number | Change amount with +/- prefix and color indication |
| Balance | Number | Running balance after transaction |
| Notes | String | Reference numbers, remarks, or descriptions |

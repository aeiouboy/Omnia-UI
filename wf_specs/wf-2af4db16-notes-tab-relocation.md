### Implementation Notes (Version 2)
- **Floating Button:** Fixed position, `bottom-6 right-6`
- **Badge:** Note count on FAB using same badge component
- **Popover:** `Popover` component from shadcn/ui
- **Auto-Navigate:** After saving via FAB, optionally switch to Notes tab
- **Hide FAB:** When Notes tab is active (avoid redundancy)

### Pros & Cons (Version 2)
| Pros | Cons |
|------|------|
| Quick access from any tab | More complex implementation |
| Full notes view in tab | Two ways to add notes |
| Non-intrusive FAB | FAB may overlap content |
| Clear note count always visible | Requires position management |

- **Sidebar Width:** `w-80` (320px) when expanded
